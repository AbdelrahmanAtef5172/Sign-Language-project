# Sign Language TTS Pipeline

## Setup

1. Install Python 3.10 from https://www.python.org/downloads/release/python-31011
   - Make sure to check "Add Python to PATH" during installation

2. Install Microsoft C++ Build Tools from https://visualstudio.microsoft.com/visual-cpp-build-tools
   - Check "Desktop development with C++" only
   - Restart your computer after installation

3. Clone the repo and open the folder in terminal, then run:
   pip install -r requirements.txt

4. Run the project:
   python main.py

## Output
The generated speech will be saved as output.wav in the project folder
